# Diabetes-CHAID-Project-1
This project analyzes a diabetes dataset and builds four different CHAID-based classification models to predict:  Diabetes Outcome  BMI Category  Glucose Level Category  Age Group  The project includes: ✔ STR files ✔ CHAID Model Outputs ✔ PPT Presentation ✔ Project Report (Word File) ✔ Dataset &amp; Code Files
